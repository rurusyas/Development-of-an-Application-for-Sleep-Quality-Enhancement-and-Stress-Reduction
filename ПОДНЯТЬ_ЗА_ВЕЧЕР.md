# поднять за вечер

## фронт (Cloudflare Pages)

build:

```
cd app && npm install && npm run build && npx vite build --mode telegram --emptyOutDir false && npx vite build --mode demo --emptyOutDir false
```

output: `app/dist`. в dist/ должны лежать `index.html`, `telegram.html`, `demo.html`.

## бэк (Railway)

репа подцеплена, в `backend/Dockerfile` уже всё. запушил — поднялось.

## бот (Railway)

второй сервис, корень `bot/`, переменные `BOT_TOKEN`, `API_URL`.

## проверить локально

```
cd app && npx vitest run && npx tsc --noEmit
cd app && rm -rf dist && npm run build && npx vite build --mode telegram --emptyOutDir false && npx vite build --mode demo --emptyOutDir false && ls dist/
```
